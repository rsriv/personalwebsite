Unnamed Audio API Project

I am currently working on using the HTML5/JavaScript Audio API create an engine to take a poorly recorded drum sound that "hears" when each drum sound (eg. hi hat, snare) is being played based on a frequency analyser. Then, where each poor quality sound has been heard, it can be replaced with a stock, studio recorded drum sound. Essentially, this will allow poor quality drum recordings to be transformed into studio sound.
Technologies used: HTML5 + JavaScript + Audio API
